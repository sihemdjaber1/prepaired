# PrepAired — AI-Matched Consulting Interview Prep

> **Practice smarter. Get matched. Land the offer.**

![Status](https://img.shields.io/badge/Status-Live-brightgreen?style=flat-square)

---

## The Problem

Breaking into top consulting firms (McKinsey, BCG, Bain, Deloitte) requires mastering case interviews. Most candidates either practice alone with no feedback, rely on expensive coaches (100-300 euros/hour), or get matched randomly with partners who do not share their target firms or level.

## The Solution

PrepAired is an AI-powered consulting interview prep platform that matches candidates with the right practice partners based on target firms, experience level, and availability.

---

## Features

- **AI Matching** — Get paired with candidates targeting the same firms at the same level
- **Firm-specific cases** — Practice with accurate McKinsey, BCG, Bain, Deloitte formats
- **Structured feedback** — Framework, analysis, and communication scoring
- **Progress tracking** — Monitor improvement across sessions

---

## How It Works

Create profile > Set target firms and level > Get matched > Practice > Get feedback

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript (vanilla)
- AI Layer: Anthropic Claude API (case generation + feedback)
- Matching: Algorithm based on firm preference + level alignment
- Deployment: GitHub Pages

---

## Target Users

- Masters / MBA students recruiting for MBB and Big 4
- Undergrads targeting strategy consulting roles
- Career changers entering consulting
